どうやってプライベートリポジトリとコラボるすですか？

ギットハブは自らのリポジトリプライベートになる設定が他のユーザーに表示されない為持っています。ですから、 URLを使って直接リポジトリサイトに行けば404エーラーが 発生します。この記事ではどうやって自分のリポジトリをプライベートにすると説明ではありません。まるでどうやってプライベートリポジトリを お好みユーザーとコラボします。

第1つ目

プライベートリポジトリの設定ページに行って。リポジトリ上のタブが見つける。この感じ：

第2つ目

設定ページで、「Manage Access（アクセスを管理）」を選んで。そして「Invite a collaborator（コラボユーザーをさっそる）」。

第3つ目

新しいウインドウが現れる。コラボの名前を入力して。そのユーザーのプロファイルが出たら選んで。

第4つ目

このままよくできました。成功を確認為青いメッセージが表せます。次はコラボユーザーの誘うE-mailを確認しないと、まだプライベートリポジトリがまだ見えません。

コラボユーザーの次元：

第5つ目

コラボユーザーとして、ギットハブのE-mailに行って、「View Invitation（リクエストを見る）」を選んで。

第6つ目

次は、ギットハブで「Accept Invitation（リクエストを受け入ります）」。

第7つ目

このままよくできました。できたら成功メッセージが表せます。今はコラボリポジトリが見えるになりました。

*注意:

リポジトリの主人として、コラボがリストから削除が出します。まずは、リポジトリの「Manage Access」設定に戻って、コラボリストからゴミバクアイコンを押して、 コラボの削除は完成になりました。

まとめ

リポジトリをプラベートをするとは大事です、もしAPI Keyとか盗作を避難する為とか頼もしい人しか誘っていけません。 コードをだれにも責任を忘れないように共有してください。
