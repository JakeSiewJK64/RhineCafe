Cherry Picking with Github Desktop

Cherry Picking is a way for developers to "pick" a commit from a branch and transfer it to another branch. This is useful if a branch contains too many mistakes or there is a significant change which might jeopardise the current code base and needed to be transferred to a branch where the code is safe to use.
