チェリーピックって何？

チェリーピックはお好みのコミットを他のブランチに移動する行動です。 チェリーピックの役に立ちポイントは自分のブランチで困ってる場合には他のブランチ に移動できること、貢献が無駄になれないということです。この記事ではどうやってギットハブデスクトップ を使ってチェリーピッくをすることを説明したいとつもりです👌。

第１つ目

ギットハブデスクトップのブランチ変更履歴に行って、お好みコミットを右クリクして、「Cherry Pick Commit」を選んで。こういう感じ：

第２つ目

新しいウインドウが現れ、お好みブランチを選ぶと聞いてになります。子の場合、好きなブランチを選んで。
*注意:
もし自分のブランチで変更が有ったら警告が表示されですが、「Stash changes and continue（変更を取り置く）」を安心して選んでください。取り置きした変更が後戻れます。 だが、もし「Stash」をする前に自分の「Stash」がまだ空いていない場合には古いバージョンが削除されます。ですから、お気おつけて選びなさい。

まとめ
